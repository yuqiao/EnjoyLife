1 日常使用
==============
1.1 每日工作备份
------------------
1. 每做完一个小节或者一个步骤：

   - git add -u  （ 若添加新文件用： git add -i )
   - git commit

2. 推送到服务器

   - git push

2 GitHub
============
2.1 创建repo
---------------
- Click https://github.com/repositories/new
- mkdir ~/life
- cd ~/life
- git init
- toutch README
- git add README
- git commit -m 'first commit'
- git remote add origin git@github.com:yuqiao/life.git
- git push origin master
